  2. HTML head section

    1. meta charset=utf-8 // specify the character that encode this html document. Broswer use the same charset to render this page. meta is unclosed tag.
    2. meta http-equiv=refresh content=30 // refresh page every 30 seconds.
    3. meta name=keywords content=tutorial,programming // used by search engines
    4. meta name=viewport content=width=device-width, initial-scale=1.0 // set the page width to be the width of browser. And set no scale.
    5. title name //define the tab name
